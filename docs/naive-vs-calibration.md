# Naive 모델 · 환자 평균 예측 · 딥러닝 모델 비교 분석

작성일: 2026-06-08  
데이터셋: VitalDB (train 1963 / val 680 / test 672 cases, 총 9,837,981 세그먼트)  
관련 문서: [evaluation_result.md](evaluation_result.md), [augmentation.md](augmentation.md)

---

## 1. 분석 목적

Calibration 기반 혈압 추정 논문들은 "환자로부터 몇 개의 기준 측정값을 얻으면 MAE가
크게 감소한다"고 보고한다. 본 분석은 이 성능 향상이 다음 두 가지 중 어느 쪽에 기인하는지
구분하기 위해 수행됐다.

- **(A) 진정한 파형 기반 개선**: PPG 파형으로부터 순간 혈압 변동을 더 정확하게 추적
- **(B) 환자 평균 보정 효과**: Calibration이 단순히 그 환자의 평균 혈압 수준을 알려줘서,
  전체 평균 예측에서 개인 평균 예측으로 baseline이 이동하는 효과

이를 위해 세 가지 예측 수준의 이론적·실제적 오차를 모두 계산하고 실제 딥러닝 모델 성능과
비교한다.

## 2. 세 가지 예측 수준 정의

### 수준 0: Naive 모델 — 전체 훈련셋 평균 예측

훈련셋 전체(5,858,103 세그먼트)의 레이블 평균값을 모든 세그먼트에 동일하게 출력한다.
**환자 개인 정보를 전혀 사용하지 않는다.**

```
SBP 예측값 = 117.91 mmHg  (모든 환자, 모든 세그먼트 동일)
DBP 예측값 =  63.25 mmHg
```

오차의 원인: 환자 간 차이(between-patient) + 환자 내 변동(within-patient)

### 수준 1: 환자 평균 예측 — 개인별 평균 혈압 출력

각 환자의 평균 혈압을 미리 알아서(혹은 몇 회의 캘리브레이션 측정으로 추정해서)
그 값을 해당 환자의 모든 세그먼트에 동일하게 출력한다. **시계열 정보는 사용하지 않는다.**

```
환자 A의 예측 = 환자 A의 SBP 평균 (예: 142 mmHg)
환자 B의 예측 = 환자 B의 SBP 평균 (예:  98 mmHg)
```

오차의 원인: 환자 내 변동만 (between-patient 차이는 이미 제거됨)

### 수준 2: 딥러닝 모델 — PPG 파형으로부터 회귀

PPG 파형으로부터 순간 SBP/DBP를 직접 예측한다. **환자 ID 정보를 입력으로 사용하지 않는다.**

오차의 원인: 파형에서 포착되지 않는 생리적 변동 + 모델 표현력의 한계

## 3. 분산 분해 (Variance Decomposition)

혈압의 총 분산은 **환자 간 분산(between-patient)** 과 **환자 내 분산(within-patient)** 으로
분해된다. 이 비율이 각 예측 수준의 이론적 한계를 결정한다.

### 계산 방법

각 환자 $i$의 세그먼트 수를 $n_i$, 환자 평균을 $\bar{y}_i$, 전체 평균을 $\bar{y}$라 하면:

$$\text{총 분산} = \underbrace{\frac{\sum_i n_i (\bar{y}_i - \bar{y})^2}{\sum_i n_i}}_{\text{환자 간 분산}} + \underbrace{\frac{\sum_i n_i \cdot s_i^2}{\sum_i n_i}}_{\text{환자 내 분산}}$$

### 결과

| 항목                   | SBP        | DBP        |
| ---------------------- | ---------- | ---------- |
| 전체 SD                | 20.14 mmHg | 11.91 mmHg |
| 환자 간 SD             | 11.03 mmHg | 7.60 mmHg  |
| 환자 내 SD (가중 평균) | 16.85 mmHg | 9.17 mmHg  |
| **환자 간 분산 비율**  | **30.0%**  | **40.7%**  |
| **환자 내 분산 비율**  | **70.0%**  | **59.3%**  |

**핵심 해석:**
- SBP 분산의 70%, DBP 분산의 59%가 **같은 환자 내 시간적 변동**에서 발생한다.
- 반대로, 환자 간 혈압 차이(고혈압/정상/저혈압 등)는 SBP 분산의 30%, DBP의 41%만 차지한다.
- 즉, 혈압 예측의 어려움은 환자마다 평균 혈압이 다른 것보다 **각 환자 안에서 혈압이
  시시각각 변하는 것**에 훨씬 더 크게 기인한다.

각 환자의 SD 분포가 예측 수준 1(환자 평균 예측)의 MAE를 결정한다.

| ----------------- | ------------- | ------------- |
| 분석 케이스 수    | 3,315         | 3,315         |
| 평균              | 16.79         | 9.20          |
| 중앙값            | 16.19         | 8.89          |
| p10               | 11.35         | 6.30          |
| p25               | 13.34         | 7.47          |

### 시각화

`data/dataset/sd_per_case.png` — 케이스를 SD 오름차순으로 정렬한 산점도.
SD = 5 mmHg 미만 케이스 비율을 기준선으로 표시했다.

### 관찰

- **DBP**: 67.5%의 환자가 SD < 10 mmHg로 상대적으로 안정적이지만, 중앙값도 여전히
  8.89 mmHg로 무시할 수 없는 변동이다.

## 5. 이론적 MAE 계산

정규분포 가정 하에 $\text{MAE} = \sigma \sqrt{2/\pi} \approx 0.798 \cdot \sigma$ 이다.
각 예측 수준의 이론적 MAE를 계산한다.

### 수준 0: Naive MAE (= 전체 SD × 0.798)

|                       | SBP                            | DBP                           |
| --------------------- | ------------------------------ | ----------------------------- |
| 전체 SD               | 20.14 mmHg                     | 11.91 mmHg                    |
| 이론적 MAE            | 20.14 × 0.798 ≈ **16.07 mmHg** | 11.91 × 0.798 ≈ **9.50 mmHg** |
| 실측 Naive MAE (test) | **15.68 mmHg**                 | **9.44 mmHg**                 |

> 이론값과 실측값이 근접하여 분포가 대략 정규분포임을 확인한다.

### 수준 1: 환자 평균 예측 MAE (= 환자 내 SD × 0.798)

환자 내 SD의 케이스 평균값과 가중 평균(세그먼트 수 기준)을 각각 이론적 MAE로 환산한다.

|                                | SBP                            | DBP                          |
| ------------------------------ | ------------------------------ | ---------------------------- |
| 환자 내 SD (가중 평균)         | 16.85 mmHg                     | 9.17 mmHg                    |
| 이론적 MAE (가중 SD 기반)      | 16.85 × 0.798 ≈ **13.45 mmHg** | 9.17 × 0.798 ≈ **7.32 mmHg** |
| 실측 환자 평균 예측 MAE (test) | **13.20 mmHg**                 | **7.08 mmHg**                |

> 이론값과 실측값의 일치가 높다. 이 값이 "환자 평균만 알고 예측했을 때"의
> 실제 오차로, **calibration의 이론적 상한 성능**이기도 하다.

## 7. Calibration 이득의 해석

### 세 단계의 오차 감소

```
Naive MAE          ─── [구간 A] ───▶  환자 평균 MAE  ─── [구간 B] ───▶  이상적 MAE
  SBP 15.68 mmHg                        SBP 13.20 mmHg                      SBP 0 mmHg
  DBP  9.44 mmHg                        DBP  7.08 mmHg                      DBP 0 mmHg

구간 A = 환자 간 분산 해소 (between-patient)
         SBP: 15.68 → 13.20 = 2.48 mmHg 감소
         DBP:  9.44 →  7.08 = 2.36 mmHg 감소

구간 B = 환자 내 변동 해소 (within-patient)
         SBP: 13.20 →  0    = 13.20 mmHg 감소 (아직 달성 불가)
         DBP:  7.08 →  0    =  7.08 mmHg 감소 (아직 달성 불가)
```

**구간 A**의 이득(~2.5 mmHg)은 **calibration이 환자의 평균 혈압 수준을 알아내는 것만으로
얻을 수 있는 최대 이득**이다. 개별 측정값 없이도 집단 정보(나이, 성별, 병력)만으로도
부분적으로 달성 가능하다.

**구간 B**의 이득(SBP ~13 mmHg, DBP ~7 mmHg)이 진정한 PPG 파형 분석의 목표치이며,
현재 딥러닝 모델들은 이 구간의 이득을 거의 달성하지 못하고 있다.

### Calibration 논문이 보고하는 이득의 위치

calibration 논문에서 "calibration 적용 전 MAE = 15~18 mmHg, 적용 후 MAE = 8~12 mmHg"
형태의 결과가 자주 보고된다. 이 이득(3~10 mmHg)은 상당 부분 **구간 A**(환자 평균 추정
효과)에 해당한다.

| Calibration 이득 구간             | 내용                       | 평가                                |
| --------------------------------- | -------------------------- | ----------------------------------- |
| Naive → 환자 평균 (구간 A)        | ~2.5 mmHg                  | 단순 개인 평균 추정만으로 달성 가능 |
| 환자 평균 → 현재 DL 모델          | SBP +0.23 / DBP −0.81 mmHg | 현재 DL 모델은 이 수준              |
| 환자 평균 → 더 낮음 (구간 B 일부) | 진정한 파형 기반 개선      | calibration 논문 주장 근거          |

calibration이 진정으로 구간 B의 이득을 제공하려면, calibration 적용 후 MAE가
**환자 평균 예측 MAE (SBP ≈ 13.2, DBP ≈ 7.1 mmHg) 이하**로 내려가야 한다.

## 8. 결론 및 시사점

### 8.1 현재 딥러닝 모델의 위치

| 항목                      | SBP             | DBP                 |
| ------------------------- | --------------- | ------------------- |
| Naive MAE (test)          | 15.68 mmHg      | 9.44 mmHg           |
| 환자 평균 예측 MAE (test) | 13.20 mmHg      | 7.08 mmHg           |
| 딥러닝 최고 MAE (test)    | **12.97 mmHg**  | **7.89 mmHg**       |
| 딥러닝이 환자 평균 대비   | −0.23 mmHg 개선 | **+0.81 mmHg 열화** |

- **SBP**: 딥러닝 모델이 환자 평균 예측을 근소하게 앞선다. 파형에서 환자 간 차이를
  포착하여 naive보다 낫고, 동시에 일부 환자 내 정보도 학습했음을 시사한다.
- **DBP**: 딥러닝 모델이 환자 평균 예측조차 넘지 못한다. 현재의 end-to-end 회귀 방식이
  DBP 환자 내 변동을 학습하기에 역부족임을 의미한다.

### 8.2 Calibration 연구에 대한 시사점

1. **구간 A 이득의 재평가**: Calibration 논문들이 보고하는 이득 중 calibration 없이
   naive에서 환자 평균까지의 ~2.5 mmHg 개선은 단순 개인 평균 추정으로 달성 가능하다.
   이를 구별하지 않은 채 "PPG calibration의 효과"로 보고하는 경우 과장될 수 있다.

2. **진정한 개선의 기준선**: Calibration이 진정한 파형 기반 개선을 제공한다고 주장하려면,
   calibration 적용 후 MAE가 **SBP ≤ 13.2 mmHg, DBP ≤ 7.1 mmHg** 이하임을 보여야 한다.
   많은 논문에서 reported된 calibration MAE가 이 기준보다 높으면, 개선의 상당 부분은
   환자 평균 추정 효과이다.

3. **DBP 개선 여지**: DBP는 환자 내 SD가 SBP보다 상대적으로 작아(중앙값 8.9 vs 16.2)
   환자 내 변동 포착이 더 중요하다. 현재 딥러닝 모델이 환자 평균 수준에도 못 미치는 DBP
   개선은 모델 구조 또는 학습 방식의 근본적 개선이 필요함을 시사한다.

### 8.3 향후 방향

- **환자 평균 예측 명시적 baseline 추가**: 평가 시 "환자 평균 예측 MAE"를 항상 보고해
  딥러닝 모델의 상대적 위치를 명확히 한다.
- **환자 내 변동 포착 실험**: 같은 환자의 이전 세그먼트 정보를 입력으로 활용하거나,
  환자 임베딩(patient embedding)을 학습하는 방식으로 환자 내 변동을 명시적으로 모델링한다.
- **ABP 파형 재구성 접근법**: SBP/DBP를 직접 회귀하는 대신 ABP 파형 전체를 재구성하면
  순간 혈압의 시계열 정보를 더 잘 포착할 가능성이 있다.
